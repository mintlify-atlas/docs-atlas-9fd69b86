## What is Tube Archivist?

Once your YouTube video collection grows, it becomes hard to search and find a specific video. Tube Archivist solves this by indexing your collection with metadata from YouTube so you can organize, search, and enjoy your archived videos offline through a convenient web interface.

## System requirements

Tube Archivist runs as a Docker Compose stack with three containers: the main application, Elasticsearch, and Redis.

| Resource | Minimum | Recommended |
|----------|---------|-------------|
| RAM | 2 GB | 4 GB+ |
| CPU | Dual core, 4 threads | Quad core+ |
| Disk | Depends on archive size | SSD for cache/index |

<Note>
  Elasticsearch requires the host kernel setting `vm.max_map_count` to be at least `262144`. See the [troubleshooting guide](/troubleshooting) for instructions on setting this value permanently.
</Note>
