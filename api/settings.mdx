## Get app configuration

`GET /api/appsettings/config/`

Returns the current application configuration.

<CodeGroup>

```bash curl
curl http://your-host:8000/api/appsettings/config/ \
  -H "Authorization: Token <your-token>"
```

```python Python
import requests

response = requests.get(
    "http://your-host:8000/api/appsettings/config/",
    headers={"Authorization": "Token <your-token>"},
)
print(response.json())
```

```javascript JavaScript
const response = await fetch(
  "http://your-host:8000/api/appsettings/config/",
  { headers: { "Authorization": "Token <your-token>" } }
);
const data = await response.json();
```

</CodeGroup>

**Response** `200 OK`

```json
{
  "subscriptions": {
    "channel_size": 50,
    "live_channel_size": 10,
    "shorts_channel_size": 10,
    "playlist_size": 50,
    "auto_start": false,
    "extract_flat": true
  },
  "downloads": {
    "limit_speed": null,
    "sleep_interval": null,
    "autodelete_days": null,
    "format": null,
    "format_sort": null,
    "add_metadata": false,
    "subtitle": null,
    "subtitle_source": null,
    "subtitle_index": false,
    "comment_max": null,
    "comment_sort": "top",
    "cookie_import": false,
    "pot_provider_url": null,
    "throttledratelimit": null,
    "extractor_lang": null,
    "integrate_ryd": false,
    "integrate_sponsorblock": false
  },
  "application": {
    "enable_snapshot": false,
    "enable_cast": false
  }
}
```

### Response fields

<ResponseField name="subscriptions" type="object">
  Controls how subscriptions are scanned.

  <Expandable title="subscriptions fields">
    <ResponseField name="channel_size" type="integer">Max number of regular videos to fetch per channel per scan.</ResponseField>
    <ResponseField name="live_channel_size" type="integer">Max live streams to fetch per channel per scan.</ResponseField>
    <ResponseField name="shorts_channel_size" type="integer">Max shorts to fetch per channel per scan.</ResponseField>
    <ResponseField name="playlist_size" type="integer">Max videos to fetch per playlist per scan.</ResponseField>
    <ResponseField name="auto_start" type="boolean">Automatically start downloading after a subscription scan.</ResponseField>
    <ResponseField name="extract_flat" type="boolean">Add videos to queue without fetching full metadata first.</ResponseField>
  </Expandable>
</ResponseField>

<ResponseField name="downloads" type="object">
  Controls download behavior.

  <Expandable title="downloads fields">
    <ResponseField name="limit_speed" type="integer">Maximum download speed in KB/s. `null` for unlimited.</ResponseField>
    <ResponseField name="sleep_interval" type="integer">Seconds to wait between downloads.</ResponseField>
    <ResponseField name="autodelete_days" type="integer">Delete watched videos after this many days. `null` to disable.</ResponseField>
    <ResponseField name="format" type="string">yt-dlp format string (e.g., `"bestvideo+bestaudio"`). `null` for default.</ResponseField>
    <ResponseField name="format_sort" type="string">yt-dlp format sort string.</ResponseField>
    <ResponseField name="add_metadata" type="boolean">Embed metadata into downloaded files.</ResponseField>
    <ResponseField name="subtitle" type="string">Subtitle language codes (comma-separated). `null` to disable.</ResponseField>
    <ResponseField name="subtitle_source" type="string">Subtitle source: `auto` or `user`.</ResponseField>
    <ResponseField name="subtitle_index" type="boolean">Index subtitle content for search.</ResponseField>
    <ResponseField name="comment_max" type="string">Max number of comments to index. `null` to disable.</ResponseField>
    <ResponseField name="comment_sort" type="string">Comment sort order: `top` or `new`.</ResponseField>
    <ResponseField name="cookie_import" type="boolean">Whether browser cookie import is enabled.</ResponseField>
    <ResponseField name="pot_provider_url" type="string">URL for a PO Token provider.</ResponseField>
    <ResponseField name="throttledratelimit" type="integer">Rate limit threshold before backing off.</ResponseField>
    <ResponseField name="extractor_lang" type="string">Force a language for yt-dlp extraction.</ResponseField>
    <ResponseField name="integrate_ryd" type="boolean">Fetch dislike counts via Return YouTube Dislike.</ResponseField>
    <ResponseField name="integrate_sponsorblock" type="boolean">Fetch SponsorBlock segments.</ResponseField>
  </Expandable>
</ResponseField>

<ResponseField name="application" type="object">
  General application settings.

  <Expandable title="application fields">
    <ResponseField name="enable_snapshot" type="boolean">Enable Elasticsearch snapshot functionality.</ResponseField>
    <ResponseField name="enable_cast" type="boolean">Enable Google Cast support.</ResponseField>
  </Expandable>
</ResponseField>

## Update app configuration

`POST /api/appsettings/config/`

Updates one or more configuration values. Supports partial updates — you only need to include the fields you want to change.

<CodeGroup>

```bash curl
curl -X POST http://your-host:8000/api/appsettings/config/ \
  -H "Authorization: Token <your-token>" \
  -H "Content-Type: application/json" \
  -d '{
    "downloads": {
      "integrate_sponsorblock": true,
      "comment_max": "100"
    }
  }'
```

```python Python
import requests

response = requests.post(
    "http://your-host:8000/api/appsettings/config/",
    headers={"Authorization": "Token <your-token>"},
    json={
        "downloads": {
            "integrate_sponsorblock": True,
            "comment_max": "100",
        }
    },
)
print(response.json())
```

```javascript JavaScript
const response = await fetch("http://your-host:8000/api/appsettings/config/", {
  method: "POST",
  headers: {
    "Authorization": "Token <your-token>",
    "Content-Type": "application/json",
  },
  body: JSON.stringify({
    downloads: {
      integrate_sponsorblock: true,
      comment_max: "100",
    },
  }),
});
const data = await response.json();
```

</CodeGroup>

Returns `200 OK` with the full updated configuration.

## Import a cookie

`PUT /api/appsettings/cookie/`

Imports a Netscape-format browser cookie string.

<ParamField body="cookie" type="string" required>
  Cookie file content in Netscape format.
</ParamField>

<CodeGroup>

```bash curl
curl -X PUT http://your-host:8000/api/appsettings/cookie/ \
  -H "Authorization: Token <your-token>" \
  -H "Content-Type: application/json" \
  -d '{"cookie": "# Netscape HTTP Cookie File\n..."}'
```

```python Python
import requests

with open("cookies.txt") as f:
    cookie_content = f.read()

response = requests.put(
    "http://your-host:8000/api/appsettings/cookie/",
    headers={"Authorization": "Token <your-token>"},
    json={"cookie": cookie_content},
)
print(response.json())
```

</CodeGroup>

Returns `200 OK` with the cookie validation object on success, `400` if the cookie is invalid.
