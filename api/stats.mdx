---
title: 'Stats'
description: 'Endpoints for library statistics, download history, and channel size rankings.'
---

The stats endpoints return aggregated metrics about your Tube Archivist library. All stats endpoints require authentication.

---

## Video statistics

`GET /api/stats/video/`

Returns aggregate statistics across all archived videos, broken down by video type and active state.

<CodeGroup>

```bash curl
curl http://your-host:8000/api/stats/video/ \
  -H "Authorization: Token <your-token>"
```

```python Python
import requests

response = requests.get(
    "http://your-host:8000/api/stats/video/",
    headers={"Authorization": "Token <your-token>"},
)
print(response.json())
```

```javascript JavaScript
const response = await fetch(
  "http://your-host:8000/api/stats/video/",
  { headers: { "Authorization": "Token <your-token>" } }
);
const data = await response.json();
```

</CodeGroup>

**Response** `200 OK`

```json
{
  "doc_count": 1420,
  "media_size": 429496729600,
  "duration": 5184000,
  "duration_str": "1440:00:00",
  "type_videos": {
    "doc_count": 1200,
    "media_size": 360000000000,
    "duration": 4320000,
    "duration_str": "1200:00:00"
  },
  "type_shorts": {
    "doc_count": 180,
    "media_size": 54000000000,
    "duration": 648000,
    "duration_str": "180:00:00"
  },
  "type_streams": {
    "doc_count": 40,
    "media_size": 15496729600,
    "duration": 216000,
    "duration_str": "60:00:00"
  },
  "active_true": {
    "doc_count": 1380,
    "media_size": 415000000000,
    "duration": 5040000,
    "duration_str": "1400:00:00"
  },
  "active_false": {
    "doc_count": 40,
    "media_size": 14496729600,
    "duration": 144000,
    "duration_str": "40:00:00"
  }
}
```

<ResponseField name="doc_count" type="integer">
  Total number of archived videos.
</ResponseField>

<ResponseField name="media_size" type="integer">
  Total disk usage in bytes across all videos.
</ResponseField>

<ResponseField name="duration" type="integer">
  Total video duration in seconds.
</ResponseField>

<ResponseField name="duration_str" type="string">
  Human-readable total duration.
</ResponseField>

<ResponseField name="type_videos" type="object">
  Aggregate stats for regular videos only.
</ResponseField>

<ResponseField name="type_shorts" type="object">
  Aggregate stats for YouTube Shorts only.
</ResponseField>

<ResponseField name="type_streams" type="object">
  Aggregate stats for live streams only.
</ResponseField>

<ResponseField name="active_true" type="object">
  Aggregate stats for videos still active on YouTube.
</ResponseField>

<ResponseField name="active_false" type="object">
  Aggregate stats for videos no longer available on YouTube.
</ResponseField>

Each `type_*` and `active_*` sub-object has the same fields: `doc_count`, `media_size`, `duration`, `duration_str`.


```javascript JavaScript

  { headers: { "Authorization": "Token <your-token>" } }
```

</CodeGroup>

**Response** `200 OK`

```json
{
  "total": {
    "duration": 5184000,
    "duration_str": "1440:00:00",
    "items": 1420
  },
  "watched": {
    "duration": 2592000,
    "duration_str": "720:00:00",
```

<ResponseField name="total" type="object">

  <Expandable title="fields">
    <ResponseField name="duration" type="integer">Total duration in seconds.</ResponseField>

    <ResponseField name="progress" type="float">Fraction of total library watched (0.0–1.0).</ResponseField>
    <ResponseField name="items" type="integer">Number of watched videos.</ResponseField>
  </Expandable>
</ResponseField>

<ResponseField name="unwatched" type="object">
  Stats for videos not yet watched. Same fields as `watched`.
</ResponseField>

## Biggest channels

`GET /api/stats/biggestchannels/`

Returns the largest channels in your library, ranked by a chosen metric.

<ParamField query="order" type="string">
  Ranking metric. One of:
  - `doc_count` (default) — by number of videos
  - `duration` — by total video duration
  - `media_size` — by total disk usage
</ParamField>

<CodeGroup>

```bash curl
curl "http://your-host:8000/api/stats/biggestchannels/?order=media_size" \
  -H "Authorization: Token <your-token>"
```

```python Python
import requests

response = requests.get(
    "http://your-host:8000/api/stats/biggestchannels/",
    headers={"Authorization": "Token <your-token>"},
    params={"order": "media_size"},
)
print(response.json())
```

```javascript JavaScript
const response = await fetch(
  "http://your-host:8000/api/stats/biggestchannels/?order=media_size",
  { headers: { "Authorization": "Token <your-token>" } }
);
const data = await response.json();
```

</CodeGroup>

**Response** `200 OK` — array of channel summary objects:

```json
[
  {
    "id": "UCxxxxxx",
    "name": "Example Channel",
    "doc_count": 142,
    "duration": 511200,
    "duration_str": "142:00:00",
    "media_size": 85899345920
  },
  {
    "id": "UCyyyyyy",
    "name": "Another Channel",
    "doc_count": 98,
    "duration": 352800,
    "duration_str": "98:00:00",
    "media_size": 53687091200
  }
]
```

<ResponseField name="id" type="string">
  YouTube channel ID.
</ResponseField>

<ResponseField name="name" type="string">
  Channel name.
</ResponseField>

<ResponseField name="doc_count" type="integer">
  Number of archived videos.
</ResponseField>

<ResponseField name="duration" type="integer">
  Total duration in seconds.
</ResponseField>

<ResponseField name="duration_str" type="string">
  Human-readable total duration.
</ResponseField>

<ResponseField name="media_size" type="integer">
  Total disk usage in bytes.
</ResponseField>

## Health check

`GET /api/health/`

Returns `200 OK` with the body `"OK"`. No authentication required. Use this for container health probes.

<CodeGroup>

```bash curl
curl http://your-host:8000/api/health/
```

```python Python
import requests

response = requests.get("http://your-host:8000/api/health/")
print(response.text)  # "OK"
```

```javascript JavaScript
const response = await fetch("http://your-host:8000/api/health/");
const text = await response.text();
console.log(text); // "OK"
```

</CodeGroup>

## Ping

`GET /api/ping/`

Verifies authentication and returns the current user and application version.

<CodeGroup>

```bash curl
curl http://your-host:8000/api/ping/ \
  -H "Authorization: Token <your-token>"
```

```python Python
import requests

response = requests.get(
    "http://your-host:8000/api/ping/",
    headers={"Authorization": "Token <your-token>"},
)
print(response.json())
```

```javascript JavaScript
const response = await fetch("http://your-host:8000/api/ping/", {
  headers: { "Authorization": "Token <your-token>" },
});
const data = await response.json();
```

</CodeGroup>

**Response** `200 OK`

```json
{
  "response": "pong",
  "user": 1,
  "version": "0.4.10",
  "ta_update": false
}
```

<ResponseField name="response" type="string">
  Always `"pong"`.
</ResponseField>

<ResponseField name="user" type="integer">
  The authenticated user's ID.
</ResponseField>

<ResponseField name="version" type="string">
  Current Tube Archivist version string.
</ResponseField>

<ResponseField name="ta_update" type="boolean">
  Whether a newer version of Tube Archivist is available.
</ResponseField>
